API Reference

Module containing models representing patients and their data.

The Model layer is responsible for the 'business logic' part of the software.

Patients' data is held in an inflammation table (2d array) where each row contains inflammation data for a single patient taken over a number of days and each column represents a single day across all patients.

def daily_max(data):
    """Calculate the daily max of a 2d inflammation data array.

    :param data: A 2d numpy array containing inflammation data
    :return: A 1d numpy array containing the max inflammation per day
    """
    return np.max(data, axis=0)

def load_csv(filename):
    """Load a Numpy array from csv

    Args:
        filename (str): path to the csv file

    Returns:
        np.ndarray: 2D array of inflammation data
    """
    return np.loadtxt(fname=filename, delimiter=',')
